What is a Finite State Machine

A finite state machine is a mathematical model of computation that describes the behaviour of a system using states, transitions, and actions.
  • States are drawn as circles, with text inside for their name
  • Transitions are drawn as arrows
  • Actions are indicated as the labels on top of the transitions

What are Discrete Finite Automata?

A discrete finite automata (DFA), used as the Default Machine on this site, is a type of finite state machine that processes a given input and accepts or rejects the input based on the given states and transition [8]. DFA are particularly useful in pattern recognition systems, examples include language recognition, compiler design, and cybersecurity. They have a binary output, True or False, and accepting states are indicated as a double circle
